# SnapSave Extension - Privacy Policy

> SnapSave is a tool designed to facilitate image downloading. We value your privacy and data security.

## Data Isolation Statement

SnapSave runs strictly client-side. We do not collect, transmit, or monitor your personal data, image URLs, or browsing histories across any remote cloud processing systems. Configured preferences are saved explicitly to your native local browser container profile.

## Granted System Permissions

* **Downloads Management:** Required to pass selected system assets safely to your local downloads directory.
* **Storage Context:** Saves your default format preference across sessions.
* **Global Host Matchers:** Allows deployment of the clean image overlay context across dynamic domains.

*SnapSave is free to use but is not open-source software. See the license in the extension for more details*
